# Abstract

Vision-language models can report objects that are contextually plausible but visually absent. We introduce COCO-Ghost, a lightweight counterfactual benchmark for measuring context-induced object hallucination in natural images. Using COCO instance segmentation masks, we remove target objects while preserving surrounding scene context and ask local vision-language models whether the removed object remains visible. We define Ghost Object Rate as the fraction of object claims that persist after object removal. We further introduce GHOST-Guard, a training-free evidence gate that accepts an object claim only when it is supported by a target crop and disappears after removal of the suspected object region. In a local Mac-feasible pilot using Ollama vision models, COCO-Ghost exposes object-class-specific context persistence, while GHOST-Guard downgrades suspicious claims to abstentions. This provides a simple, interpretable, black-box-compatible reliability layer for vision-language systems.
