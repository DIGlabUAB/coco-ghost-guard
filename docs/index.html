COCO-Ghost: Auditing Whether VLM Object Claims Survive Object Removal

Authors

Anusha Kanagala, M.S. · Dhiral Panjwani, M.S. · Rubin Pillay, M.D. · Leon Jololian, Ph.D. · Sandeep Bodduluri, M.S., Ph.D.

Project page · June 2026 · open code, cached outputs, and reproducible local runs

COCO-Ghost asks a blunt visual question: if the target object is removed by a conservative occlusion, does a VLM still say it is visible? GHOST-Guard is a black-box evidence gate that abstains when the claim survives removal.

Code Results Artifact audit Reproduce
The benchmark in one loop

A claim is trusted only if it disappears when its visual evidence is removed

COCO-Ghost builds three views of the same instance. GHOST-Guard accepts object presence when original and crop agree, then abstains if the masked counterfactual still triggers the same object claim.

Main finding

Clean occlusion removes the silhouette objection; context persistence remains

We moved away from blur masks after visual inspection showed that blurred segmentation regions can preserve object shape. The scaled result uses padded bounding-box occlusion with solid local-mean fill and reports all headline metrics on 200/200 clean-removal cases.

Qwen2.5-VL 7B clean ghost rate
51.6%

99 of 192 originally recognized objects were still claimed after clean occlusion.

Qwen2.5-VL 3B / LLaVA 7B
47.5 / 33.7%

The effect is not isolated to one local VLM family, though LLaVA has weaker original recognition.

Accepted detected ghost claims
0.0%

GHOST-Guard converts detected persistence cases to UNSURE; it is an abstention wrapper, not a model fix.

Important reading Some failures are exactly what a strong scene prior looks like: a player without equipment, a sink scene without a cup, a shelf after a bottle is occluded. That is the point of the benchmark, not a flaw to hide. The page below shows both successes and failures.

1Qualitative evidence: successes and failures

The examples below use the same Qwen2.5-VL 7B decision rule as the headline result. A success means the model answers NO after occlusion and the guard accepts the original object claim. A failure means the model still answers YES after clean occlusion, so the guard abstains.

2Method: ask whether evidence survives removal

Three views per COCO instance

Original asks whether the object is visible in the full image. Crop checks local visual support. Occluded removes a padded target-object bounding box using solid local-mean fill.

Guard rule

accept iff
original = YES
crop     = YES
masked   = NO

The intervention is intentionally conservative. It is not a photorealistic inpainting claim; it is an occlusion counterfactual for testing whether object-presence judgments depend on visible object evidence.

4Where persistence concentrates

The category pattern is not uniform. Some categories are heavily scene-prior driven across models, while others are easier for the model to reject after occlusion. This is why the benchmark is more useful as a diagnostic than as a single leaderboard number.

5Why the guard is useful in applications

GHOST-Guard is useful when a system needs to decide whether to trust an object-presence claim from a black-box VLM. It does not require gradients, logits, model internals, or retraining. It asks for localized evidence and then checks whether the claim disappears when the evidence is removed.

Table 2. Baseline safety comparison. Crop-gating helps but still accepts many claims that survive removal.

In practical systems, this is an abstention layer: route uncertain object claims to a human, another detector, or a higher-cost verification path rather than letting a VLM silently assert unsupported presence.

6Artifact audit and the blur-mask caveat

The first expanded runs used segmentation blur. Visual inspection showed that blur could preserve the object's silhouette, especially for elongated or high-contrast objects. Those results are now treated as an ablation and motivation for the stricter scaled protocol.

Scope COCO-Ghost should be read as a counterfactual stress test for object-presence claims, not as proof that every masked-image YES is a hallucination in the human sense.
