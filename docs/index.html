COCO-Ghost: Auditing Whether VLM Object Claims Survive Object Removal

A controlled counterfactual benchmark and black-box abstention rule for measuring context-induced object persistence in local vision-language models.

Anusha Kanagala, M.S. · Dhiral Panjwani, M.S. · Stephanie M. Aguilera, B.S. · Abhiraj Pudhota, M.S. · Kelly Powell, M.B.A. · Danish Murad, M.S. · Rubin Pillay, M.D. · Leon Jololian, Ph.D. · Sandeep Bodduluri, M.S., Ph.D.

Abstract

Vision-language models often answer object-presence questions by combining visual evidence with strong scene priors. COCO-Ghost evaluates this behavior with object-level counterfactuals: an object must be recognized in the original image and in its crop, then the same object is occluded and queried again. If the model continues to answer YES after clean target removal, the claim is counted as a ghost.

On 200 manually audited COCO validation instances, three local VLMs retain removed-object claims in 33.7-51.6% of originally recognized cases. The headline analysis uses padded bounding-box occlusion with solid local-mean fill, and reports results separately on cases labeled as clean removals.

Contributions
  1. A 200-sample audited counterfactual suite with artifact labels.

Method

Counterfactual object-presence evaluation

The experiment separates three questions that are often conflated: can the model identify the object, can it recognize local object evidence, and does the claim disappear when the evidence is removed?

Qualitative evidence

Successes, failures, and category-level structure

We show both favorable and unfavorable cases. A success flips to NO under occlusion; a failure remains YES and is treated as an abstention case.

Deployment interpretation

GHOST-Guard is an abstention rule, not a model repair

The guard does not claim to fix the VLM or explain its internals. It is a black-box reliability layer for object-presence claims: accept claims that disappear after target removal, and abstain when the claim survives a clean counterfactual. In a deployed system, abstentions can be routed to an object detector, a stronger model, or human review.

Artifact audit

The intervention matters

Early blur masks were informative but visually confounded: blurred segmentation regions can preserve target shape. The headline result therefore uses padded bounding-box occlusion with solid local-mean fill. This choice is less naturalistic, but it makes the main claim harder to dismiss as residual silhouette leakage.

Scope: COCO-Ghost is a counterfactual stress test for object-presence claims. It does not prove that every masked-image YES is a human-level hallucination, and it does not replace broader VLM robustness evaluation.
